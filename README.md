# ArirangStudio  
A browser-based audio instrument simulator built using HTML, CSS, and JavaScript.

## Project Overview
ArirangStudio is a lightweight web application that allows users to simulate musical instruments directly in the browser. It focuses on simplicity, accessibility, and ease of customization.
